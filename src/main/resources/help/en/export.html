Export to braille file.

In order to export the document to a braille file you must take into account several things:

  • The final device for the exported file.
  • The language
  • The math coding

When the Export option in the File menu is selected, three options are shown:

  • UTF: this option is usually for braille displays. The file contents are braille symbols in UTF-8 coding.
  • bfr: this file type is for braille embossers that need this format.
  • ban: exports a file with contents that follows the instructions given at Documento técnico V 1: Tabla ANSI española para braille computarizado. downloaded from https://www.once.es/servicios-sociales/braille/comision-braille-espanola/documentos-tecnicos/documentos-tecnicos-relacionados-con-otras-materias (this link opens in a browser)
