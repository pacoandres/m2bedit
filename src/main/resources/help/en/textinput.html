Text input and translation.

For writing text and maths you must use LaTex commands. These commands are restricted by the Braille code limitations and the skills of an average secondary level student

Text format commands.

By now, only three format commands are available:
  • \textbf{bold text}
  • \textit{italic text}
  • \underline{underlined text}

Each command translation depends on the language braille rules as described in the tables of the liblouis library. So, in some languages these commands can do nothing, or may give the same result.

You can see the coding capabilities for the desired language in the menu option Available languages

in the Help menu.

Math commands.

For inserting math commands you should have a math environment, and it can be done by two ways:
  • Enclosing the expression between dollar signs ($ecuation$). In this case the math is aligned with the text : $\frac{x+1}{x-1}$ = x+1 x-1
  • Enclosing between \[ and \]. This way writes the math in its own line, without anything else
About the LaTex math commands there's not so much to say about it. There is a lot of documentation about them on the internet, so here we are going to show the most significant for secondary level.
  • Superscript: $x^{n}$ = x n
  • Subscript: $x_{i}$ = x i
  • Fraction: $\frac{a}{b}$ = a b
  • Square root: $\sqrt{x+1}$ = x + 1
  • Root: $\sqrt[n]{x+1}$ = x + 1 n
  • Vector: $\vec{v}$ = v →
  • Segment: $\overline {AB}$ = A B
With these commands we can write the quadratic equation formula as:

$\frac{-b \pm \sqrt{b^2 - 4ac}{2a}$ = − b ± b 2 − 4 a c 2 a

Translation of Mathematical expressions

For translating mathematical expressions the library mathcat is used.

The available languages inside mathematical expressions and the available coding for these expressions can be seen at Math translations in the Help menu

The encoding to be used must be the one recommended by the institution that carries out Braille standardization in your country.
