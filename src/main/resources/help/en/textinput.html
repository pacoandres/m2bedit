Text input.

For writing text and maths you must use LaTex commands. These commands are restricted by the Braille code limitations and the skills of an average secondary level student

Text format commands.

By now, only three format commands are available:
  • \textbf{bold text}
  • \textit{italic text}
  • \underline{underlined text}
Each command translation depends on the language Braille rules. For example, in spanish Braille there's only one type of emphasis so the three commands are translated to the same emphasis.

Math commands.

For inserting math commands you should have a math environment, and it can be done by two ways:
  • Enclosing the expression between dollar signs ($ecuation$). In this case the math is aligned with the text : $\frac{x+1}{x-1}$ = x+1 x-1
  • Enclosing between \[ and \]. This way writes the math in it's own line, without anything else
About the LaTex math commands there's not so much to say about it. There is a lot of documentation about them on the internet, so here we are going to show the most significant for secondary level.
  • Superscript: $x^{n}$ = x n
  • Subscript: $x_{i}$ = x i
  • Fraction: $\frac{a}{b}$ = a b
  • Square root: $\sqrt{x+1}$ = x + 1
  • Root: $\sqrt[n]{x+1}$ = x + 1 n
  • Vector: $\vec{v}$ = v →
  • Segment: $\overline {AB}$ = A B
